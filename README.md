# ContaBanco

## Descrição
Este projeto é uma aplicação simples em Java que permite aos usuários inserirem dados de uma conta bancária através do terminal. Após a inserção dos dados, a aplicação exibe uma mensagem de agradecimento ao usuário, contendo as informações da conta inseridas.

## Como usar
1. Clone este repositório para o seu ambiente local.
2. Abra o terminal e navegue até o diretório do projeto.
3. Compile o código-fonte com o comando `javac ContaTerminal.java`.
4. Execute o programa com o comando `java ContaTerminal`.
5. Siga as instruções apresentadas no terminal para inserir os dados da conta bancária.

## Exemplo de interação

1. Programa: "Número: ".
Usuário: 123456789 (depois ENTER para o próximo campo).

2. Programa: "Agencia: "
Usuário: 123-4 (depois ENTER para o próximo campo).

3. Programa: "Titular: "
Usuário: João da Silva (depois ENTER para o próximo campo).

4. Programa: "Saldo: "
Usuário: 1000.00 (depois ENTER para finalizar).


## Exemplo de saída

"Olá João da Silva, obrigado por criar uma conta em nosso banco, sua agência é 123-4, conta 123456789 e seu saldo 1000.00 já está disponível para saque".
